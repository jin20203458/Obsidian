
# 🔄 2. 고정점 정리 (Fixed-Point Theorem)

## 📌 고정점이란?

**고정점(Fixed Point)**은 **함수에 자기 자신을 넣어도 변하지 않는 점**입니다.

> 수학적으로:  
> $$ f(x) = x $$  
> 를 만족하는 점 ( x )를 **고정점**이라 합니다.

---

## 💡 간단한 예시

```cpp
// f(x) = x/2 + 1 함수
f(0) = 1    // 0은 고정점이 아님
f(1) = 1.5  // 1도 고정점이 아님
f(2) = 2    // ✅ 2가 고정점! f(2) = 2
```

---

## 📈 그래프로 보면

- **y = f(x)** 곡선과 **y = x** 직선이 **만나는 점**이 고정점입니다.

---

## 🔍 왜 고정점이 중요한가?

> **"반복해도 더 이상 변하지 않는 안정된 상태"**를 나타냅니다.

---

## 🔄 반복 적용 예시

```cpp
// 계속 함수를 적용해보기
x₀ = 어떤 값
x₁ = f(x₀)  
x₂ = f(x₁)
x₃ = f(x₂)
...
xₙ = f(xₙ₋₁)

// 고정점에 도달하면:
xₙ₊₁ = f(xₙ) = xₙ  // 더 이상 변화 없음
```
