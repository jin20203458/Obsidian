
# 📐 1. 격자이론 (Lattice Theory)

## 격자란 무엇인가?

**격자(Lattice)**는 **특별한 순서 관계**를 가진 집합입니다.  
이를 **높이 관계**로 생각할 수 있습니다.

```
        최고점 (Top) ⊤
       ↙         ↘
    중간점들 ...
       ↘         ↙
        최저점 (Bottom) ⊥
```

---

## 💡 일상 생활 예시: 정보의 정확성

- **⊤** = "완전히 확실함"
- **중간** = "어느 정도 확실함"
- **⊥** = "전혀 모름"

예시:

- **Bottom (⊥)**: "아무것도 모름"
- **중간**: "x는 확실하지만 y는 모름"
- **Top (⊤)**: "모든 것을 확실히 알고 있음"

---

## 🖥 InitBeforeUseCheck의 격자 예시

3개의 변수 `(x, y, z)`의 초기화 상태를 격자로 표현:

```
⊤ = [1,1,1]   // 모든 변수가 초기화됨
      ↙
[1,1,0]       // x, y만 초기화됨
      ↙
[1,0,0]       // x만 초기화됨
      ↙
⊥ = [0,0,0]   // 아무것도 초기화 안됨
```

---

## 🔹 격자의 중요한 연산: Meet (∧)

- **Meet** = "공통점 찾기" = "더 보수적인 쪽 선택"

```cpp
// 두 상태의 교집합
[1,1,0] ∧ [1,0,1] = [1,0,0]
//   ↑       ↑        ↑
//  x,y만   x,z만     x만 (공통분모)
```

**직관**: "둘 다 확실한 것만 남기기"
