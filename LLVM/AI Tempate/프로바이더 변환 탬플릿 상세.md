> "아래 Clang-tidy C++ 코드를 분석해서 `CheckerProviderService`에 추가할 C# 코드를 작성해줘.
> 
> 1. 변수명은 체커 이름을 카멜 케이스(camelCase)로 변환해서 작성할 것.
>     
> 2. `Options.get`으로 가져오는 옵션들을 분석해서 `BoolOptionItem`, `UnsignedOptionItem`, `StringOptionItem` 등으로 매핑할 것.
>     
> 3. 각 옵션 뒤에 `// C++ Default: [값]` 주석을 반드시 달 것.
>     
> 4. `OnlyMainFile`, `IgnoreSystemHeaders`, `IgnoreMacroExpansions` 같은 공통 옵션이 코드에 없더라도, C# 코드에는 기본적으로 `true`로 추가할 것 (필요 시)."
>

// C++ Source (Example)
void MyAwesomeCheck::storeOptions(ClangTidyOptions::OptionMap &Opts) {
    Options.store(Opts, "AnalyzeTemplate", AnalyzeTemplate);
    Options.store(Opts, "MaxDepth", MaxDepth);
}

MyAwesomeCheck::MyAwesomeCheck(StringRef Name, ClangTidyContext *Context)
    : ClangTidyCheck(Name, Context),
      AnalyzeTemplate(Options.get("AnalyzeTemplate", true)),
