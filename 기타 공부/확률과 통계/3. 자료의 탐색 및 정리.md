# 3장: 자료의 탐색 및 정리 핵심 정리

## 1. 자료의 구조와 형태

- **개체**: 표에서 각 행(예: 한 학생의 성적 데이터)이 하나의 개체
    
- **변수**: 각 열(예: 과목명, 점수 등)이 변수
    
- **관측값**: 한 개체(행)에 해당하는 모든 변수값
    
- **변수값**: 특정 변수(열)에 대한 값들
    
- **자료집합**: 모든 자료값의 집합
    
- **일변량 자료**: 변수가 1개인 자료
    
- **다변량 자료**: 변수가 2개 이상인 자료
    

## 2. 측정의 척도

- **명목척도**: 단순 분류(예: 성별, 출생지, 직업). 숫자는 단지 구분용, 사칙연산 불가
    
- **순서척도**: 순서가 있는 분류(예: 등급, 선호도, 암의 단계). 순서만 의미, 간격·비율 의미 없음
    
- **수치척도**: 숫자 간 차이가 의미 있음
    
    - _구간척도_: 임의의 0(예: 섭씨온도)
        
    - _비율척도_: 절대 0(예: 길이, 무게)
        
- **이산형 자료**: 정수값(예: 학년, 성별)
    
- **연속형 자료**: 연속적인 값(예: 키, 몸무게)
    
- **범주형 자료**: 구간별로 나눈 자료(예: 나이를 10대, 20대 등으로 구분)
    
- **독립변수/종속변수**: 한 변수가 다른 변수에 영향을 주면, 영향을 주는 쪽이 독립변수, 받는 쪽이 종속변수
    

## 3. 이산형 자료의 표와 그래프

- **막대그래프**: 명목/순서형 자료의 도수(빈도)나 비율을 막대 형태로 시각화
    
- **원그래프**: 전체 중 각 범주가 차지하는 비율을 원의 부채꼴로 표현
    

## 4. 수치형 자료의 표와 그래프

- **도수분포표**: 자료를 구간(계급)별로 나누고, 각 구간에 속하는 자료의 개수(도수), 비율(상대도수), 누적도수 등을 표로 정리
    
    - 계급 수는 보통 6~14개, 구간은 겹치지 않고 등간격이어야 함
        
- **히스토그램**: 도수분포표를 막대그래프로 표현. X축은 구간, Y축은 도수(또는 비율)
    
- **줄기와 잎 그림**: 원자료의 값을 잃지 않고 분포를 표현. 줄기는 큰 자리수, 잎은 작은 자리수
    
- **상자-수염 그림(박스플롯)**: 자료의 분포, 중심, 퍼짐, 이상치 등을 한눈에 보여줌
    
    - 박스: 1사분위수(Q1)~3사분위수(Q3)
        
    - 중앙값: 박스 내부 선
        
    - 수염: 최대/최소값(이상치 제외)
        
    - 이상치: Q1-1.5_IQR, Q3+1.5_IQR 바깥의 값
