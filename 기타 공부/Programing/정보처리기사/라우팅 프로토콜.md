| 약어    | 풀네임 (Full Name)                            | 의미 및 설명                                                                   |
| ----- | ------------------------------------------ | ------------------------------------------------------------------------- |
| RIP   | Routing Information Protocol               | 라우팅 정보 프로토콜. 거리 벡터 알고리즘 기반으로 홉 수 (hop count) 기준 최단 경로 탐색. 소규모 네트워크에서 사용됨. |
| OSPF  | Open Shortest Path First                   | 개방 최단 경로 우선. 링크 상태 라우팅 프로토콜로, 다익스트라 알고리즘을 사용해 더 정확한 최단 경로 탐색.             |
| EIGRP | Enhanced Interior Gateway Routing Protocol | 향상된 내부 게이트웨이 라우팅 프로토콜. 시스코 독점 하이브리드 프로토콜로 빠른 컨버전스 특징.                     |
| BGP   | Border Gateway Protocol                    | 경계 게이트웨이 프로토콜. 인터넷 대형 네트워크 간 라우팅에 사용되는 경로 벡터 프로토콜.                        |
