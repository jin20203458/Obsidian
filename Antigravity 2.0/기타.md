## 대화방 삭제


### 대화방 정리 유틸리티 사용 방법

1. **안티그래비티(또는 연동된 IDE 플러그인)를 완전히 종료**합니다.
2. 터미널(Command Prompt 또는 PowerShell)을 열고 아래 명령어를 실행합니다:
   ```cmd
   python C:\Users\user\.gemini\antigravity\conversations\clean_convo.py
   ```
3. 실행하시면 스크립트가 모든 대화방 데이터베이스를 스캔하여 아래 예시와 같이 대화 미리보기 목록을 번호순으로 보여줍니다.
   ```text
   =========================================================================
    [번호] 대화방 미리보기 (ID)
   =========================================================================
    [0] 프로젝트 전체 점검하며 더미코드나 다이어트 가능한 코드 확인하고... 
        (ID: 710ffb3e-0ec5-464b-802c-e061dfc790c3)

    [1] 다른 대화방 첫 질문 내용 샘플...
        (ID: aed4eb63-bc35-44c0-a2c3-40fdca429a3f)
   =========================================================================
    [A] 모든 대화방 한꺼번에 초기화(전체 삭제)
    [Q] 프로그램 종료
   =========================================================================
   삭제할 대화방 번호 또는 명령을 입력하세요:
   ```
4. 삭제하고 싶으신 대화방의 **번호를 입력하고 Enter**를 치거나, 전체를 다 지우고 싶으실 경우 **`A`를 입력**하시면 해당 대화방의 `.db` 파일 3개 및 `brain` 폴더가 일괄 삭제 처리됩니다.

이 도구를 사용하시면 대화방의 UUID를 하나하나 분석하실 필요 없이 직관적이고 빠르게 꼬인 대화창을 지우실 수 있습니다.
