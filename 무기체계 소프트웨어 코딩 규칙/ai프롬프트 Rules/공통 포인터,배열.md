
### [포인터-가] 널 포인터 역참조 금지
- checkerName: DereferenceChecker, NullDereferenceGuardCheck
- category: Pointer
- severity: CRITICAL
- summary: "포인터를 역참조(*p, p->)하기 전에는 반드시 널(NULL) 여부를 확인해야 합니다."
- violation: `r = *p;` ([translate:널 체크 없음])
- correction: `if (p != NULL) { r = *p; }`

### [포인터-나] 댕글링 포인터 생성 금지 (넓은 스코프 할당)
- checkerName: StackAddrEscapeChecker
- category: Pointer
- severity: ERROR
- summary: "함수 내 지역 변수의 주소를 함수 외부의 전역 변수나 더 넓은 스코프를 가진 변수에 할당할 수 없습니다."
- violation: `int a; pi = &a;` (`pi`[translate:가 전역 변수일 때])
- correction: [translate:더 넓은 스코프로 지역 변수 주소를 노출하지 않습니다.]

### [포인터-다] 댕글링 포인터 생성 금지 (지역 변수 주소 반환)
- checkerName: StackAddrEscapeChecker
- category: Pointer
- severity: ERROR
- summary: "함수가 종료되면 사라지는 지역 변수의 주소를 반환할 수 없습니다."
- violation: `int a = 0; return &a;`
- correction: `static int a = 0; return &a;` [translate:또는 동적 할당 사용]

### [배열-라] 배열 범위 초과 접근 금지 (Out-of-Bounds Access)
- checkerName: ArrayBoundCheckerV2, ArrayIndexBoundCheck
- category: Array
- severity: ERROR
- summary: "배열에 접근할 때 유효한 인덱스 범위를 벗어나서는 안 됩니다."
- violation: `int a[10]; r = a[10];` ([translate:유효 인덱스는 0~9])
- correction: `int a[10]; r = a[9];`

### [포인터-마] 널 포인터 산술 연산 금지
- checkerName: NullPointerArithmeticCheck
- category: Pointer
- severity: MAJOR
- summary: "널(NULL) 포인터에 대해 증가(++), 감소(--) 등의 산술 연산을 수행할 수 없습니다."
- violation: `if (q == NULL) { ++q; }`
- correction: `if (q != NULL) { ++q; }`
