

### [식별자-가] extern과 static 혼용 금지
- checkerName: NoMixedLinkageCheck
- category: Linkage
- severity: ERROR
- summary: "하나의 식별자에 extern(external linkage)과 static(internal linkage)을 동시에 선언할 수 없습니다."
- violation: |
    extern int number;
    static int number; // 모순된 선언
- correction: |
    extern int number; // 일관된 선언 사용

### [식별자-나] External 식별자 유일성
- checkerName: UniqueExternalNameCheck
- category: Linkage
- severity: ERROR
- summary: "External linkage를 갖는 함수나 객체의 이름은 전체 프로그램(모든 번역 단위)에서 유일해야 합니다."
- violation: |
    // file1.c
    int g_var;
    // file2.c
    float g_var; // 중복 정의
- correction: |
    // file1.c
    int g_var;
    // file2.c
    float g_var_file2; // 유일한 이름 사용

### [식별자-다] External 식별자 타입 일치
- checkerName: MatchingExternalTypeCheck
- category: Linkage
- severity: ERROR
- summary: "External linkage를 갖는 객체의 선언과 정의는 반드시 동일한 데이터 타입을 가져야 합니다."
- violation: |
    // file1.c
    extern int g_var;
    // file2.c
    float g_var = 0.0; // 타입 불일치
- correction: |
    // file1.c
    extern float g_var;
    // file2.c
    float g_var = 0.0; // 타입 일치

### [식별자-라] 변수 섀도잉(Shadowing) 금지
- checkerName: NoVariableShadowingCheck
- category: Scope
- severity: WARNING
- summary: "내부 스코프에서 외부 스코프의 변수와 동일한 이름의 변수를 선언하여 가려서는 안 됩니다."
- violation: |
    int x = 1;
    if (true) {
        int x = 2; // 외부 변수 x를 가림
    }
- correction: |
    int x = 1;
    if (true) {
        int inner_x = 2; // 다른 이름 사용
    }
