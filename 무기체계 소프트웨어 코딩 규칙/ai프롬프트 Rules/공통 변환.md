
### [변환-가] 숫자 표현 범위 초과 할당 금지
- checkerName: NoOutOfRangeAssignmentCheck
- category: Type Conversion
- severity: ERROR
- summary: "변수 할당 시 해당 데이터 타입이 표현할 수 있는 값의 범위를 초과해서는 안 됩니다 (Integer Overflow)."
- violation: `short x = 40000;` (short 범위 초과)
- correction: `long x = 40000;`

### [변환-나] 가변인수 포맷-타입 일치
- checkerName: VarargsFormatCheck, clang-analyzer-valist.UnmatchedFormatSpec
- category: Variadic Functions
- severity: ERROR
- summary: "printf 등 가변인수 함수에서 포맷 지정자(%d, %s 등)와 실제 전달 인자의 타입은 반드시 일치해야 합니다."
- violation: `printf("%s", 10);` (%s에 int 전달)
- correction: `printf("%d", 10);`

### [변환-다] 가변인수 포맷-개수 일치
- checkerName: VarargsFormatCheck, clang-analyzer-valist.UnmatchedFormatSpec
- category: Variadic Functions
- severity: ERROR
- summary: "printf 등 가변인수 함수에서 포맷 지정자의 개수와 실제 전달 인자의 개수는 반드시 일치해야 합니다."
- violation: `printf("%d %d", 10);` (인자 1개 부족)
- correction: `printf("%d", 10);` 또는 `printf("%d %d", 10, 20);`

### [변환-라] 데이터 손실 형 변환 금지
- checkerName: RepresentableCastCheck, clang-analyzer-core.RangeChecks
- category: Type Conversion
- severity: MAJOR
- summary: "값이 더 큰 타입에서 작은 타입으로 형 변환 시 데이터가 손실될 위험이 있는 코드를 작성하지 않습니다."
- violation: `signed int si = 200; signed char sc = si;` (데이터 손실)
- correction: `if (si <= 127) { sc = si; }` (범위 체크 후 변환)

### [변환-마] 음수를 unsigned로 변환 금지
- checkerName: ConversionChecker, clang-analyzer-core.Bitwise-operations
- category: Type Conversion
- severity: MAJOR
- summary: "음수 값을 unsigned 타입으로 변환하면 의도치 않은 큰 양수 값으로 변경(wrap-around)되므로 금지합니다."
- violation: `unsigned int ui = -1;`
- correction: `signed int si = -1;`

### [변환-바] char와 wide char 문자열 혼용 금지
- checkerName: NoMixedCharWideStringCheck
- category: String
- severity: WARNING
- summary: "char* 기반 문자열과 wchar_t* 기반 와이드 문자열을 직접 연결하거나 혼용할 수 없습니다."
- violation: `char* s = "Hello" L"World";`
- correction: `char* s1 = "Hello" "World";`, `wchar_t* s2 = L"Hello" L"World";`

### [변환-사] const 한정자 제거 캐스팅 금지
- checkerName: PointerCvQualifierDropCheck, clang-analyzer-security.insecureAPI.constCast
- category: Type Casting
- severity: ERROR
- summary: "포인터 형 변환(cast)을 통해 const 한정자를 제거하여 읽기 전용 데이터를 수정 가능하게 만들어서는 안 됩니다."
- violation: `const int* p; int* q = (int*)p;`
- correction: "const 한정자를 제거하는 캐스팅을 사용하지 않습니다."

### [변환-아] volatile 한정자 제거 캐스팅 금지
- checkerName: PointerCvQualifierDropCheck, clang-analyzer-security.insecureAPI.volatileCast
- category: Type Casting
- severity: ERROR
- summary: "포인터 형 변환(cast)을 통해 volatile 한정자를 제거하면 컴파일러 최적화로 인해 예기치 않은 동작이 발생할 수 있습니다."
- violation: `volatile int* p; int* q = (int*)p;`
- correction: "volatile 한정자를 제거하는 캐스팅을 사용하지 않습니다."
