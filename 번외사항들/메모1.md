
## 단일어

- **Sentinel (센티넬)** → 이미 쓰고 계신 이름. “파수꾼, 감시자”라는 의미로 **지켜본다, 감시한다**는 이미지가 강합니다. 국방·산업 규격 검증 솔루션에 딱 맞는 상징성.
    
- **Armored (아머드)** → “갑옷을 입은, 무장된”이라는 뜻. **보호·방어** 이미지를 직관적으로 줍니다. 다만 단어 자체가 일반적이라 독창성은 조금 약할 수 있습니다.
    
- **Ironclad (아이언클래드)** → “철갑”이라는 뜻. **절대 흔들리지 않는, 확고한**이라는 비유적 의미로도 많이 쓰입니다. 묵직하고 강력한 울림. 이미 글로벌 SaaS(계약 관리 플랫폼) 브랜드로 쓰이고 있긴 하지만, 업종이 달라 구분 가능성은 있습니다.
    
- **Regulus (레귤러스)** → 라틴어로 ‘작은 왕’, 별 이름으로도 유명. **권위·고전적 울림**이 있습니다. 다만 이미 여러 업계(제약, 소프트웨어 등)에서 사용 중이라 차별화 전략이 필요합니다.
    
- **Bastion (바스티온)** → “요새, 보루”라는 뜻. **최후의 방어선**이라는 상징성이 강합니다. 다만 Microsoft Azure의 네트워크 서비스 이름으로도 쓰이고 있어 IT 업계에서는 혼동 가능성이 있습니다.
    
- **Anvil (엔빌/엔벌)** → “모루”라는 뜻. **강철을 두드려내는 기반**이라는 산업적 이미지가 있습니다. 짧고 단단한 울림, 제조·산업 규격 검증과 잘 어울립니다.
    
- **Rampart (램파트)** → “성벽, 방벽”이라는 뜻. **방어·보호**의 상징. 흔히 쓰이지 않아 독창성이 있고, 발음도 힘이 있습니다.

## 합성어
### Sentinel 계열

- **Sentinelix** (센티넬릭스) → Sentinel + Matrix, “규정을 지키는 체계”
    
- **Sentivus** (센티버스) → Sentinel + Virtus(라틴어 ‘힘, 덕’)
    
- **Sentra** (센트라) → Sentinel + Central, “중심에서 지켜보는”
    

### Armored / Ironclad 계열

- **Armoryx** (아머릭스) → Armored + Matrix, “무장된 체계”
    
- **Cladion** (클래디언) → Ironclad + Bastion, “철갑 요새”
    
- **Ironsure**(아이언 슈어) → Ironclad + Assure, “철같이 확실한”
    

### Regulus / Bastion 계열

- **Reguard** (리가드) → Regulus + Guard, “규정의 수호자”
    
- **Bastiron** (배스티런) → Bastion + Iron, “철의 보루”
    
- **Regnix** (레그닉스) → Regulus + Nexus, “규정과 연결된 중심”
    

### Anvil / Rampart 계열

- **Anviora** (앤비오라) → Anvil + Aurora, “새로운 빛을 두드려내는 기반”
    
- **Ramforge** (램포지) → Rampart + Forge, “성벽 같은 단단한 단조”
    
- **Anvex** (엔벡스) → Anvil + Vertex, “정점의 기반”
