## **데이터 플로우 (상세):**
```
사용자: DataGrid에서 진단 항목 선택
      ↓
OnSelectedDiagnosticChanged(DiagnosticItemEx) 트리거
      ↓
IsAIAnalyzed 확인 → false면 자동 분석
      ↓
AnalyzeWithAIAsync(diagnostic, CancellationToken.None)
      ↓
_aiAnalysisService.AnalyzeDiagnosticAsync(...)
      ↓
프롬프트 구성:
  - 체커 이름: "defence-style-variable-init"
  - 코드 스니펫: 파일에서 추출한 코드
  - 진단 메시지: "Variable not initialized"
  - 규칙 정보: GetRuleContextForChecker()
      ↓
ChatClient.CompleteChatAsync(messages)
      ↓
OpenAI API 호출 (HTTP POST)
      ↓
응답 수신: Markdown 텍스트
      ↓
diagnostic.AIAnalysis = responseText
diagnostic.IsAIAnalyzed = true
      ↓
UI 갱신 (INotifyPropertyChanged)

```

## **전체 분석 (일괄)**
```
사용자: [전체 AI 분석] 버튼 클릭
      ↓
AnalyzeAllAsync() 실행
      ↓
_analysisCancellationTokenSource = new CancellationTokenSource()
var cancellationToken = _analysisCancellationTokenSource.Token
      ↓
foreach (var diagnostic in unanalyzedItems)
{
    ┌──────────────────────────────────────┐
    │ if (cancellationToken.IsCancellationRequested) break;
    │       ↓
    │ await AnalyzeWithAIAsync(diagnostic, cancellationToken)
    │       ↓
    │ _aiAnalysisService.AnalyzeDiagnosticAsync(...)
    │       ↓
    │ OpenAI API 호출
    │       ↓
    │ 응답 수신 → diagnostic.AIAnalysis 업데이트
    │       ↓
    │ successCount++ 또는 failureCount++
    │       ↓
    │ await Task.Delay(500, cancellationToken)  // Rate limiting
    └──────────────────────────────────────┘
}
      ↓
전체 완료 또는 취소 메시지 표시

```
## **취소 플로우**
```
사용자: [AI 분석 취소] 버튼 클릭
      ↓
CancelAnalysis() 실행
      ↓
_analysisCancellationTokenSource.Cancel()
      ↓
cancellationToken.IsCancellationRequested = true
      ↓
┌─ AnalyzeAllAsync foreach 루프에서 감지
│      ↓
│  if (cancellationToken.IsCancellationRequested) break;
│
├─ Task.Delay(500, cancellationToken)
│      ↓
│  TaskCanceledException 발생
│      ↓
│  catch → break;
│
└─ AnalyzeDiagnosticAsync 내부
       ↓
   cancellationToken.ThrowIfCancellationRequested()
       ↓
   OperationCanceledException 발생
       ↓
   catch → break;

```

##  **결과 표시 플로우**
```
사용자: [AI 분석 결과 보기] 버튼 클릭
      ↓
OpenAIAnalysisWindow() 실행
      ↓
if (SelectedDiagnostic == null || !IsAIAnalyzed)
    → MessageBox 표시 후 리턴
      ↓
var viewModel = new AIAnalysisViewModel(SelectedDiagnostic, _aiAnalysisService)
      ↓
var window = new AIAnalysisWindow { DataContext = viewModel }
      ↓
window.ShowDialog()
      ↓
AIAnalysisWindow.xaml 렌더링:
  - MdXaml.MarkdownScrollViewer
  - Markdown="{Binding Diagnostic.AIAnalysis}"
      ↓
MdXaml이 Markdown → FlowDocument 변환
      ↓
사용자에게 표시
```
