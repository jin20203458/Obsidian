##  .clang-tidy 파일 탐색 규칙

Clang-tidy는 다음 순서로 `.clang-tidy` 파일을 찾습니다:

## 탐색 순서

1. **분석 대상 파일이 있는 디렉토리**

2. 상위 디렉토리로 계속 올라가며 탐색

3. 루트 디렉토리(`C:\`) 또는 홈 디렉토리까지

4. 파일을 찾으면 **거기서 탐색 중단**


## 예시 구조
```scss
C:\
└─Users\
  └─user\
    └─Desktop\
      ├─ .clang-tidy         ← ③ 여기까지 탐색
      └─ErrorTest\
        ├─ .clang-tidy       ← ② 여기 있으면 사용
        └─ErrorTest\
          ├─ .clang-tidy     ← ① 여기를 먼저 찾음
          └─ test1.cpp       ← 분석 대상

```
